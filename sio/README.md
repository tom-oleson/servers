# sioecho
Demonstrates a simple serial monitor

Requires libcm_64.a from project common

<pre>
usage: sioecho [-v] port

-v            Output version/build info to console
</pre>

Example:

<pre>
$ sioecho /dev/ttyACM0
</pre>
